## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

# visit at 
```bash
localhost:3000
```

## Running directly with Docker
```bash
$ docker-compose up nestapp
```

## Installation

```bash
$ npm install
```

## Running the app
```bash
$ nest start
```

## Test
```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
